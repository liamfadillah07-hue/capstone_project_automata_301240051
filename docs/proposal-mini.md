## 1. Rancangan Sistem

dibangun murni dengan HTML/CSS/JavaScript (tanpa framework) supaya dapat
dideploy sebagai situs statis di GitHub Pages tanpa proses build.

## 2. Diagram State / Grammar Acuan
Sertakan di sini contoh diagram DFA sederhana dan contoh grammar CFG yang
dipakai untuk pengujian awal (lihat `/tests/test-cases.md`).

## 3. Tech Stack
- Frontend: HTML5, CSS3 (custom, tanpa framework), JavaScript (ES6+, vanilla)
- Visualisasi: SVG digambar manual lewat DOM API (`src/js/svg.js`)
- Hosting: GitHub Pages, domain kustom `.my.id`
- Version control: Git/GitHub, commit bertahap per modul
